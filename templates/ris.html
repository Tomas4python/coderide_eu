Reverse Image Search (RIS)

View code on Github

I have been working on a Reverse Image Search (RIS) project as part of my learning journey. This project uses EfficientNet models and TensorFlow to analyze and compare images. By leveraging pretrained convolutional neural networks, the system extracts image features and performs similarity searches with K-Nearest Neighbors (KNN) and Cosine Similarity. This setup ensures accurate image matching and has been quite insightful for my understanding of these technologies.

About the Notebooks

In the notebooks available in my GitHub repository, I cover the complete workflow for developing the Reverse Image Search (RIS) application. This includes downloading the 'Caltech101' image dataset and initializing models with EfficientNetB0 and EfficientNetV2 for comparison. The notebooks guide you through extracting image features using these pretrained CNNs and performing similarity searches with K-Nearest Neighbors (KNN) and Cosine Similarity. Additionally, I use PCA and t-SNE for visualizing the data, providing a comparative analysis of the extracted features. Below I’ve included several visualizations from the notebooks. The first image shows query results, displaying both the query images and the returned matches. The second image is a t-SNE visualization of Caltech 101 features, offering a scatter plot view. The third image showcases 2000 examples from the Caltech101 dataset, also plotted using t-SNE.
