Web Scraping App: Simplifying Computer Shopping

View code on Github

In today's fast-paced world, finding the right computer at the best price can often become an overwhelming task. Sifting through multiple online stores, comparing prices, reading descriptions - the entire process can be daunting. This is precisely where the "Web Scraping App" comes into play, streamlining the experience and giving users a seamless way to find their perfect computer deal.

App Features

  1. Two-Window Interface:
    • Search Window: Users are presented with pre-defined search strings via buttons, an input field for custom search terms, and logos of nine prominent computer stores in Lithuania, providing a straightforward search experience.
    • Results Window: A dynamic window displaying search outcomes â€“ computer descriptions, corresponding website links, and their prices. Results are systematically sorted from the lowest to the highest price. Multiple result windows can be initiated, giving users an invaluable tool for comparison.
  2. Simplified Shopping Experience: The primary goal is to alleviate the tediousness of browsing multiple websites. Now, finding and comparing computer prices is just a click away.
  3. Learning Undertaking: The focal point of this project was to master the intricacies of tkinter, BeautifulSoup, and Selenium. Additionally, understanding the web structures of various online stores was crucial.

Code Organization

For this iteration, I opted for a single module approach. However, to ensure clarity and maintainability, functions were grouped under relevant classes.
