Discovering the Essence of this Website

View code on Github

This platform was conceptualized and brought to life with three distinct intentions:

  1. Getting to know Django 4: One of my primary goals was to acquaint myself with the Django 4 module. Furthermore, deploying this website on a server not inherently tailored for Django or Python presented its own set of unique challenges.
  2. A Dive into Web Development: This endeavor provided an excellent opportunity to get skills and understanding of the foundational web languages: HTML, CSS, and JavaScript.
  3. To introduce My Coding Journey: Through this site, I aim to share my programming learning trajectory and the milestones I've reached, in a format that's both engaging and easily digestible.

The driving forces behind my journey into the realm of Django, HTML, CSS, and JavaScript have been two amazing publications shown in the photo below:

  • Django for Beginners: Build websites with Python & Django by William S. Vincent
  • Sams Teach Yourself HTML, CSS, and JavaScript All in One, Third Edition by Jennifer Kyrnin and Julie C. Meloni

In my journey to understand web development, I delved into Django 4 and grasped its structure. I honed my skills in HTML for content layout and experimented with CSS for design enhancements. Additionally, I gained a deeper understanding of JavaScript functionalities.

One of the pivotal experiences was deploying a Django site on a dedicated server. I familiarized myself with Gunicorn and Nginx, and even ventured into setting up Ubuntu.

While I've acquired extensive knowledge, I wanted this site to be streamlined and lightweight. My previous experiences with tools like Selenium and BeautifulSoup, as well as publishing on platforms like Joomla and WordPress, made me realize the importance of having a solid foundation in HTML, CSS, and JavaScript.
